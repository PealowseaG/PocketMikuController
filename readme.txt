This repository is the controller for Pocket Miku with raspberry pi + LCD HAT.
Including Python3 codes, shell script for Main menu of Raspberry Pi and documents.
Main functions are change instrument sound, set Pocket Miku voice code from Japanese Hiragana text file and change effects (reverb, chorus and variation).
Demo movie: https://youtu.be/K2TxZqKXfN8
